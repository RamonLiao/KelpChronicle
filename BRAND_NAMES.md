# 品牌名稱建議

核心概念：具備長期、可驗證記憶的研究 Agent。每次執行會先回憶既有研究、計算新舊差異，只分析新增資訊，再把版本化研究 artifact 存入 Walrus，並將內容 hash 錨定到 Sui。

## 1. TideScribe

`Tide` 代表持續變化的生態與資訊潮汐；`Scribe` 是負責觀察、整理及保存紀錄的人。

名稱呈現一個持續追蹤市場變化、只記錄新發展的研究 Agent。簡短、好唸，也能延伸成 `TideScribe Memory`、`TideScribe Reports` 等產品模組。

## 2. KelpChronicle

`Kelp` 是會持續生長並形成大型生態系的海藻；`Chronicle` 代表按時間累積的可信紀錄。

海藻森林很像 Agent 的長期記憶：每次研究增加新的分支，但仍與既有知識彼此連結。適合強調跨 session 累積、可探索的研究歷史。

## 3. CurrentRecall

`Current` 同時代表水流、當前狀態與最新資訊；`Recall` 對應 Agent 在每次研究前取回既有記憶。

名稱完整表達產品的核心循環：先 recall 過去，再找出 current delta。定位清楚，是由目前代號平順升級成正式品牌的選項。

## 4. NautilusDelta

`Nautilus` 的殼室會隨成長逐層增加，象徵保存每次研究形成的歷史記憶；`Delta` 代表新舊資料之間的差異。

名稱最精準地結合「累積記憶」與「只研究增量」兩項差異化能力，也很適合螺旋時間線、版本環及 run comparison 等視覺設計。

## 5. PelagicMemory

`Pelagic` 指開放海域，帶有持續探索廣大未知領域的意象；`Memory` 直接表明產品不是一次性搜尋工具，而是持久記憶系統。

適合未來從單一研究 Agent 擴張到共享 namespace、多 Agent 協作及 Agent memory marketplace，品牌範圍比目前研究情境更寬。

## 推薦排序

1. **NautilusDelta** — 最能表達「逐次累積記憶＋只研究增量」的技術差異。
2. **TideScribe** — 最像成熟的使用者品牌，簡潔、有畫面且容易記憶。
3. **CurrentRecall** — 與現有 Recall 代號銜接最自然，產品機制也很清楚。
4. **KelpChronicle** — Walrus 生態感與長期知識網絡意象最強。
5. **PelagicMemory** — 適合未來擴張為通用 Agent memory infrastructure。
