KelpChronicle

A persistent, verifiable memory for autonomous agents — stored on Walrus, anchored on Sui, grown as a living kelp forest.

The problem

Agents have amnesia.

Transient

Restart the agent and its memory vanishes.

Locked-in

Trapped in a centralised Web2 vector database.

Unverifiable

You can't prove it wasn't tampered with.

The fix

Memory that outlives the agent.

Memory lives on Walrus, via MemWal. Every run's content hash is anchored on-chain on Sui through recall::attestation.

It survives restarts. It moves between machines. Anyone can audit it.

The metaphor

We grow it as a
living kelp forest.

Every strand is a research run. Every bud is a finding. The structure is driven entirely by real data — let me show you.
